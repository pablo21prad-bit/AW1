  • Comandos
  • git init
    • Inicializa un nuevo repositorio Git en el directorio actual.
  • git clone [URL]
    • Clona un repositorio remoto en tu máquina local.
    • descargar un proyecto y su historial de versiones.
  • git add [archivo(s)]
    • Agrega archivos específicos al área de preparación (staging area) para ser confirmados (committed).
    • podes agregar un archivo específico, varios archivos o todos los archivos modificados.
  • git commit -m "[mensaje]"
    • crea un nuevo commit con los cambios en el área de preparación y un mensaje descriptivo.
    • el mensaje es una breve descripción de los cambios realizados.
  • git status
    • Muestra el estado actual del repositorio, incluyendo archivos modificados, no rastreados y en el área de preparación.
    • te informa sobre los archivos que han cambiado y cuáles están listos para ser confirmados.
