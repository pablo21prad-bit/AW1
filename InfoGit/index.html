¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite a los desarrolladores rastrear y gestionar los cambios en su código fuente a lo largo del tiempo. Fue creado por Linus Torvalds en 2005 para ayudar en el desarrollo del kernel de Linux, y desde entonces se ha convertido en una herramienta esencial para la colaboración en proyectos de software.

¿Qué es GitHub?

GitHub es una plataforma de alojamiento de código fuente basada en la web que utiliza Git como sistema de control de versiones. Fue fundada en 2008 y se ha convertido en una de las plataformas más populares para el desarrollo colaborativo de software. GitHub permite a los desarrolladores almacenar, compartir y colaborar en proyectos de código abierto y privado.
